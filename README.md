# Excel to JSON Converter for Flutter Localization

A powerful cross-platform tool (Android, Windows, Web, macOS, Linux, iOS) designed to streamline the Flutter localization workflow. It converts Excel spreadsheets containing translations into the JSON format required by Flutter's localization systems (like `easy_localization` or the native `flutter_localizations`).

## üöÄ Features

*   **Excel to JSON Conversion**: Takes an `.xlsx` file and generates individual `.json` files for each language column.
*   **Drag & Drop Support**: Simply drag your Excel file onto the app window to start the process.
*   **Multi-Platform**: Works seamlessly on Desktop (Windows, macOS, Linux), Web, and Mobile (Android, iOS).
*   **Open Source**: Contribute and improve the tool!
